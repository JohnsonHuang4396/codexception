# 2026-03-04 codexception-mvp Design

## Context
目标是在 Codex 环境中复现 Claudeception 的核心价值：把一次性解决过程沉淀为可复用技能。

## Scope
- 仅建立目录、规则、模板、路线图。
- 不实现自动触发机制。

## Design Decision
- 优先文档驱动，降低早期复杂度。
- 以 `AGENTS.md` 作为“强制复盘”入口。
- 以 `skills/skill-template.md` 作为统一输出格式。

## Success Criteria
- 项目可直接用于人工沉淀技能。
- 新成员可以在 10 分钟内理解流程并产出首个 skill 草稿。
